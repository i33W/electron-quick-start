### Electron 일렉트론?
- Electron은 JavaScript, HTML 및 CSS를 사용하여 데스크톱 애플리케이션을 구축하기 위한 프레임워크입니다.
- Chromium 및 Node.js를 바이너리에 포함함으로써 \
 Electron을 사용하면 하나의 JavaScript 코드베이스를 유지 관리하고 \
 Windows, macOS 및 Linux에서 작동하는 크로스 플랫폼 앱을 만들 수 있습니다.
- 기본 개발 경험이 필요하지 않습니다.
